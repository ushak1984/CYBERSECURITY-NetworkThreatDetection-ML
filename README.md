# Cybersecurity Threat Detection with Machine Learning

This capstone project investigates how machine learning can support cybersecurity threat detection using both attack classification and anomaly detection.

The project focuses on identifying malicious network or system behavior from public cybersecurity datasets, while also highlighting practical challenges such as class imbalance, false alarms, and dataset limitations.

## Project Goal

The main goal was to evaluate whether machine learning models can help detect:

- Known attack patterns using supervised classification
- Unusual or unseen behavior using anomaly detection
- Dataset quality issues that affect real-world cybersecurity modeling

## Files

- `README.md` : project overview
- `Python Notebooks/EDA/` : data cleaning, preprocessing, and exploratory analysis notebooks
- `Python Notebooks/ML MODELS/` : supervised classification and anomaly detection notebooks
